# Intelligent-Traffic-Scene 
# 交通场景智能应用软件

## 1️⃣ 功能列表
✅ 利用YOLOv3实现车辆、行人、红绿灯检测  
✅ 利用HSV原理检测红绿灯颜色  
✅ 利用HyperLPR实现车牌检测  
✅ 目标分类统计、路段车流量统计，并将结果存储在Excel文件中  
⭕ 更多高级的行为识别功能等待开发......

## 6️⃣ 作者的一些话
2020年4月中旬，我参加了“中国软件杯”大学生软件设计大赛，当初是抱着学习的心态报名的，也没想到真的能够做出一个勉强称之为“软件”的东西，但不管怎么说也是一个具有交互功能的软件程序，这也是我第一次花费如此多的心血在一个项目上。虽然说我们的软件目前仍然只是一只“缝合怪”，但整体的软件结构我花费了不少心思去构思、实现，我也能够感受到自己编程能力的提高。在我敲下这些文字的时候我也进入了软件设计大赛的决赛，在此非常感谢我的两位舍友，给我提供灵感、技术以及文档方面的支持，这是我们共同开发的软件。
